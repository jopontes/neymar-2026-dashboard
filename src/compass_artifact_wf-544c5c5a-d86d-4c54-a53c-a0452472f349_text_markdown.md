# Estatísticas detalhadas de 13 brasileiros de 2018/19 a 2025/26

**Nota metodológica inicial.** Este relatório consolida dados extraídos de **FBref, Transfermarkt, Sofascore, FotMob, Wikipedia (tabelas compiladas de Soccerway/oficiais), Understat, StatMuse, FootyStats, Flashscore e sites oficiais de clubes** (Real Madrid, FC Barcelona, Brentford FC, Club Brugge). Sempre que o FBref oferece cobertura Opta (Top-5 ligas europeias + Primeira Liga + Eredivisie + Bélgica + Série A brasileira desde ~2022), foram priorizadas métricas avançadas (xG, xA, passes progressivos, SCA, dribles). **Para Série B brasileira, Championship inglês, Bulgarian First League, Saudi Pro League e seleção brasileira, dados avançados (xG/xA/SCA/passes progressivos) não têm cobertura consistente e são reportados como "N/A".** Dados de 2025/26 são **parciais** (corte ~abril/2026). Valores com "~" são estimativas cruzadas; valores sem prefixo vêm de fontes oficiais/FBref. "empty" = jogador ainda não havia estreado profissionalmente. A correção do Igor Thiago (nunca jogou Bayer Leverkusen — trajetória real: Cruzeiro → Ludogorets → Club Brugge → Brentford) e de Luís Henrique (o jogador nascido em 2001 NÃO é o campeão da Libertadores 2024 pelo Botafogo — esse foi Luiz Henrique, homônimo diferente) são detalhadas ao fim.
